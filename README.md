# Nex-Up Precios - Servicios Remotos

Web estática para mostrar precios de servicios remotos.

## Archivos
- `index.html` - Página principal
- `styles.css` - Estilos

## Subir a GoDaddy

### Opción 1: FTP (FileZilla)
1. Descarga FileZilla (gratis)
2. Conecta usando las credenciales FTP de GoDaddy
3. Arrastra los archivos a la carpeta `public_html`

### Opción 2: Panel de GoDaddy
1. Ve a Mis Productos → Web Hosting → Gestionar
2. Busca "Archivos" o "File Manager"
3. Sube los archivos a `public_html`

### Opción 3: GitHub Pages (gratis)
1. Sube este repo a GitHub
2. Ve a Settings → Pages
3. Configura como GitHub Pages
